= Controls =
Xbox Controller is recommend
other DirectInput controllers can be used but may have unusual bindings

	Escape - Quit Game

	Menu
	****
		Keyboard
		--------
			Arrow Keys - Navigate
			Enter      - select option
			
		Controller
		----------
			D-Pad - Navigate
			Start - select option
			
	Game
	****
		Keyboard
		--------
			Player 1
			""""""""
				A, D  - Move
				W     - Jump
				Space - Punch
			Player 2
			""""""""
				(have numlock on)
				Numpad 4, Numpad 6 - Move
				Numpad 8           - Jump
				Right Arrow Key    - Punch
			
		Controller
		----------
			Left Stick - Move
			A          - Jump
			X          - Punch
			Start      - Restart level
			
			
= Cheat Codes / Debugging tools =

	F1     - Player 1 instant win
	Enter  - Restart level
	Delete - Toggle Debug Draw
